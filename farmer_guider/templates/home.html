About This Project

1. Objective

This project is designed to assist farmers in making informed decisions about the most suitable crops to plant based on specific environmental conditions.

2. Environmental Factors Considered

By inputting key factors such as nitrogen, phosphorus, potassium levels, temperature, humidity, pH, and rainfall, the system leverages a K-Means clustering algorithm to predict the optimal crop selection.

3. Crop Prediction Process

The algorithm clusters the data into groups and assigns the input data to the cluster that best matches its characteristics. As a result, the farmer receives a precise crop recommendation that aligns with the current environmental conditions.

4. Results Display

Once a crop is predicted, the result is displayed with both the crop's name in English and Hindi, along with a corresponding image of the crop.

5. User Interface

The platform includes an intuitive interface where users can enter the environmental factors easily. The system processes the data, performs the prediction, and presents the results clearly and concisely.

6. Data Storage and Analysis

The entire process is supported by a robust backend system that saves the prediction data for future reference and analysis.
